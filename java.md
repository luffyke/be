### ORM
ORM全称Object Relational Mapping，既是对象关系映射，主要是把关系型数据库的表和行映射成对象，方便进行面向对象编程，从效果上说，它其实是创建了一个可在编程语言里使用的--“虚拟对象数据库”。
- Mybatis, [官网](http://www.mybatis.org/mybatis-3/), [Mybatis产品](http://blog.mybatis.org/p/products.html)
